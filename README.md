## 프로젝트 목표

### redux-toolkit으로 작업하기

💡 **상태 관리를 효율적이게 할 수 있는 방법이 뭐가 있을까 고민했다.**

플레이리스트를 바로 저장하거나 삭제할 수 있게 redux-toolkit으로 작업하는 것을 목표로 두었다.

### API 요청하기

💡 **유튜브 API를 받아와 연결해보기**

음악 관련 API를 찾아보다가 유튜브 API가 하루 할당량은 다른 음악 API들보다 적지만 사용하기에 제일 깔끔하기에 API를 받아왔다.
